Multi OAuth2 Proxy

Design System & Component Catalog

This design system provides a comprehensive set of UI components and styles for the Multi OAuth2 Proxy authentication service. Built with Tailwind CSS 4, it features an adaptive color scheme that seamlessly transitions between light and dark themes. All components are optimized for accessibility and user experience, following modern web design principles. The catalog below demonstrates each component in various states and configurations, showing how they work together to create a cohesive authentication interface.

OTP Input with Validation

One-Time Password input field that validates the format (12 uppercase letters/digits). The border and background turn green when a valid code is entered. Try entering: ABCD EFGH 1234

<input type="text" class="input" placeholder="XXXX XXXX XXXX" maxlength="14" style="text-align: center; font-family: 'Courier New', monospace; font-size: 1.125rem; font-weight: 600; letter-spacing: 0.05em; background-color: var(--color-bg-muted); border: 2px solid var(--color-border-default); transition: border-color 0.2s ease, background-color 0.2s ease;"> <script> const otpInput = document.getElementById('your-otp-input'); function validateOTP(value) { const cleaned = value.replace(/[^A-Z0-9]/gi, '').toUpperCase(); return cleaned.length === 12 && /^[A-Z0-9]{12}$/.test(cleaned); } otpInput.addEventListener('input', function() { if (validateOTP(this.value)) { this.style.borderColor = 'var(--color-success)'; this.style.backgroundColor = 'color-mix(in srgb, var(--color-success) 10%, var(--color-bg-muted))'; } else { this.style.borderColor = 'var(--color-border-default)'; this.style.backgroundColor = 'var(--color-bg-muted)'; } }); </script>
