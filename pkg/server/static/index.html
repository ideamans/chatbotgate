Multi OAuth2 Proxy

Design System & Component Catalog

This design system provides a comprehensive set of UI components and styles for the Multi OAuth2 Proxy authentication service. Built with Tailwind CSS 4, it features an adaptive color scheme that seamlessly transitions between light and dark themes. All components are optimized for accessibility and user experience, following modern web design principles. The catalog below demonstrates each component in various states and configurations, showing how they work together to create a cohesive authentication interface.
